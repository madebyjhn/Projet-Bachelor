## Prérequis

- Node.js installé
- Git
- VS Code ou tout IDE compatible

---

## Commands to clone and launch the project

```bash
# Clone the repository
git clone https://github.com/madebyjhn/Projet-Bachelor.git

# Get into the project folder
cd Projet-Bachelor

# Install dependencies
npm install

# Run the server
npm run dev
```
